# Instagram Reels Feature Testing During a Call

## Objective
  Verify the functionality of the Reels feature during an Instagram call.

## Preconditions
1. The tester has an Instagram account.
2. A second user is available to participate in the Instagram call.
3. Both users have a stable internet connection.

## Test Scenario
1. Verify that the internet connection is stable.
2. Open the Instagram application.
3. Start a call with the second user.
4. Check that the main call features work correctly.
5. Test the Reels feature in detail during the call.

## Expected Result
  The Reels feature should work smoothly without lag. The application should display relevant videos for both users, and both video Reels and photo posts should work correctly. Navigation between posts should be smooth, without freezing or repeated content.
